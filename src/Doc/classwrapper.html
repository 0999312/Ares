Detailed Description

template<class OBJ, class TYPE, class EXT, class TYPEEXT>
class wrapper< OBJ, TYPE, EXT, TYPEEXT >

This class acts as a wrapper class around a game object to provide transparent access to all members, including the ones provided by Ares-extensions. In addition, it adds various helpers.

Priority-order: 1. Ares Extension 2. Game Object 3. Type Extension 4. Game Type

NOTE: This is designed for the "four big" object types, Aircraft, Buildings, Infantry and Vehicles. It will not work on other types.
