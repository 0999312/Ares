Parameters:
[in] Bullet The projectile that hits the victim.
Return values:
true An abduction was performed. It should be assumed that the target is gone from the map at this point.
false No abduction was performed. This can be because the weapon is not an abductor, or because an error occurred. The target should still be on the map.
